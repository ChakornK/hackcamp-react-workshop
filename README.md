### Prerequisites

Make sure you have [Node.js](https://nodejs.org/en/download/) installed.

Ensure that you have corepack enabled. You can run the following command to enable corepack:

```bash
corepack enable
```

### Installing dependencies

```bash
yarn install
```

### Running the app

```bash
yarn dev
```
